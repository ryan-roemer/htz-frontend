@haaretz/fela-utils

General, non-ui utils for building Fela apps and components in the Haaretz stack

Installation

yarn add @haaretz/fela-utils

createRenderer

packages/components/fela-utils/src/createRenderer.js

A factory for creating a custom Fela renderer, tailored to an app's needs

createRenderer(options: object?): function
Parameters
options (object? = {}) an options object.
Name Description
options.isDev boolean (default process.env.NODE_ENV!) Manually override the node environment settings. Can be useful, e.g., for testing. Set according to NODE_ENV by default
options.isRtl boolean? Set the renderer to handle RTL layouts
options.selectorPrefix boolean (default 'htz-') A string to namespace all class names with
Returns
function: A fela createRenderer function
