### Credit Component

Creates a HTML `address` tag, for displaying article author/credit.
The credit can be linked to author page if url is passed to Component.

```jsx
<div dir="rtl">
  <Credit 
    name="דני בר-און"
  />
</div>
```

With link:
```jsx
<div dir="rtl">
  <Credit 
    name="דני בר-און" 
    url="/misc/writers/1.1799644"
  />  
</div>
```
