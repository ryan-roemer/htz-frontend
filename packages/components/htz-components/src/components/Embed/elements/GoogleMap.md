### GoogleMap component

An example for Google map street view.

```jsx static
<GoogleMap
  embedType = 'streetView'
  settings = {{
    heading: 'none',
    coordinates: '32.05216,34.772368',
    language: 'en',
    satellite: 'false',
    pitch: '00',
    fov: '20',
  }}
  caption = 'street view'
  credit = 'GoogleMap'
/>
```
