### StandardAudio component

An example for a SoundCloud playlist.

```jsx static
<StandardAudio
  embedType = 'playlist'
  content = "https://soundcloud.com/maikbaptist-two/sets/bad-religion-plus-coversongs"
  caption = "playlist"
  credit = "SoundCloud"
/>
```
